Le dossier `{{directoryName}}` contient une liste de ficher de bande-dessinée :

{{files}}

Chacun de ces fichiers doit ętre renomé suivant le format suivant :

<serie> #<numéro> - <titre>.<extension>

La serie est le nom de la série de bande-dessinée, le numéro est le numéro du tome dans la série, et le titre est le titre du tome. L'extension doit ętre conservée.

Le nom du dossier `{{directoryName}}` est également le nom de la série de bande-dessinée.

Tu retournera une liste de fichiers renommés dans un tableau contenant les colonnes suivantes :
| OldName | NewName |
